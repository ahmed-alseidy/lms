## Medium Priority (Competitive Advantages)

### 6. Affiliate Marketing

- Affiliate links, commission tracking, payouts, dashboard, reports, referral codes.

### 7. Drip Content / Scheduled Release

- Lesson release dates, time-based unlock, prerequisites (complete X before Y), course start dates.

### 8. Downloadable Resources

- PDFs per lesson, resource library per course, download tracking, file type limits.

### 9. Live Sessions / Webinars

- Zoom/Meet integration, scheduled live classes, recordings, attendance, live chat.

### 10. Advanced Quiz Features

- Question banks, essay (manual grading), fill-in-blank, matching, explanations after submit, retake limits, pass/fail thresholds.

### 11. Student Messaging

- Teacher–student DMs, in-app and email notifications, message history.

### 12. Course Prerequisites

- Require Course A before B, show prerequisites on course page, auto-unlock when met.
